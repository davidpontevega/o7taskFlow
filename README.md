# 🚀 o7TaskFlow

Sistema moderno de gestión de tareas y flujos de trabajo diseñado para equipos, productividad personal y automatización de procesos.

## ✨ Características

- Gestión de tareas
- Organización por estados y prioridades
- Gestión de usuarios
- Notificaciones y seguimiento
- Dashboard de productividad
- API REST

## 🛠️ Tecnologías

- Frontend: React / Next.js, TailwindCSS
- Backend: Node.js, Express
- Base de datos: PostgreSQL / MySQL
- ORM: Prisma

## ⚙️ Instalación

git clone https://github.com/davidpontevega/o7taskFlow.git  
cd o7taskFlow  
npm install  

## ▶️ Ejecución

npm run dev  

## 🔑 Variables de entorno

PORT=3000  
DATABASE_URL=  
JWT_SECRET=  

## 📡 Endpoints

GET /tasks  
POST /tasks  
PUT /tasks/:id  
DELETE /tasks/:id
